# McKinsey Chart Design Skill

> *FT/Economist editorial charts — designed by consultants, generated by AI.*

## When to Use

Activate this skill when:

- User asks to generate data visualization charts or line charts
- User mentions "chart prompt", "Midjourney chart", "图表提示词"
- User needs trend visualization in McKinsey/FT/Economist editorial style
- User provides time-series data and wants publication-quality visualization

## Design System

### Core Principles

| Principle | Description |
|-----------|-------------|
| **Minimalism** | Maximum data-ink ratio, remove all non-essential elements |
| **Annotation-driven narrative** | Labels tell the story, not legends |
| **Direct labeling** | Labels at line endpoints, no legend boxes |
| **Focal emphasis** | One bold black "protagonist" line, others are supporting |
| **Whitespace** | Generous breathing room |

### Color Palette — Cool Editorial

| Role | Color | Hex | Line Width |
|------|-------|-----|------------|
| **Focal Line** | Bold Black | #000000 | ~3px |
| Series 2 | Dark Navy | #003399 | ~2px |
| Series 3 | Cyan | #00BFFF | ~2px |
| Series 4 | Teal | #008080 | ~2px |
| Series 5 | Light Cyan | #66CCCC | ~1.5px |
| Series 6 | Lavender | #9999CC | ~1.5px |
| Series 7 | Purple | #660099 | ~1.5px |

### Fixed Parameters

| Parameter | Value |
|-----------|-------|
| Aspect Ratio | 16:9 (`--ar 16:9`) |
| Font | Arial (all text) |
| Background | Pure white (#FFFFFF) |
| Gridlines | None |
| Axes | Minimal, key integers only |
| Legend | None (direct labeling) |
| Midjourney Params | `--style raw --v 6` |

## Workflow

### Step 1: Information Gathering
Extract: topic, line count, time range, focal line, highlight events

### Step 2: Data Research (if needed)
Use web search to find real trend data and key events

### Step 3: Design Planning
Assign colors, determine focal line, plan highlight zones

### Step 4: Generate 4 Prompt Versions
- **Version A**: Detailed (recommended)
- **Version B**: Standard
- **Version C**: Minimal
- **Version D**: Chinese

### Step 5: Deliver as .md file
Sections: Data context, Design plan, 4 prompts, Usage tips

## Limitations

- Midjourney cannot precisely control data values — charts are "visual illustrations"
- For exact data, recommend Python/Excel generation instead
- 3-7 lines optimal; >7 becomes visually crowded
