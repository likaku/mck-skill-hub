## What Problem Does This Solve?

Beautiful editorial charts take hours to design in Illustrator or Figma. This skill generates Midjourney prompts that produce publication-quality charts in the McKinsey / Financial Times / Economist style — cold-palette, annotation-driven, with direct labeling and focal emphasis.

## Quick Start

```bash
cp -r . ~/.claude/skills/mck-chart-design/
```

## Usage

```
"Generate a Midjourney chart prompt showing global AI investment trends
from 2020 to 2026, with China as the focal line, comparing US, EU, Japan, and Korea."
```

The skill will research real data, design the chart specification, and output 4 versions of Midjourney prompts (detailed, standard, minimal, Chinese).
